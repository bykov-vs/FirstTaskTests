Кейс 1. Создание клиента
  Предусловие:
  1. Открыть браузер
  2. Перейти на страницу входа: https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager
  3. Клиента с параметрами Test, Test, Test не существует в таблице
  
  Шаги:
  1. Нажать на кнопку Add Customer
  2. В появившейся форме заполнить поля First Name, Last Name, Post Code значением Test
  3. Нажать на кнопку Add Customer, находящуюся под формой
  
  Ожидаемый результат:
  1. В появившемся диалоговом окне содержится текст 
     "Customer added successfully with customer id :" и его числовой идентификатор (id) после двоеточия 
=========================================================================================================
Кейс 2. Создание дубликата клиента
  Предусловие:
  1. Открыть браузер
  2. Перейти на страницу входа: https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager
  3. В таблице существует клиент Harry Potter E725JB

  Шаги:
  1. Нажать на кнопку Add Customer
  2. В появившейся форме заполнить поля First Name, Last Name, Post Code соответствующими значениями
  Harry, Potter, E725JB
  3. Нажать на кнопку Add Customer, находящуюся под формой

  Ожидаемый результат:
  1. В появившемся диалоговом окне содержится текст
     "Please check the details. Customer may be duplicate."
=========================================================================================================
Кейс 3. Сортировка клиентов по имени по возрастанию
  1. Открыть браузер
  2. Перейти на страницу входа: https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager
  3. Нажать на кнопку customers
  4. В таблице есть как минумум две записи
  
  Шаги:
  1. Нажать на текст-ссылку First Name в первом столбце, первой строке таблицы на экране
  2. Повторно нажать на текст-ссылку First Name
  
  Ожидаемый результат:
  1. Клиенты отсортированы по лексикографическому порядку по возрастанию
=========================================================================================================

Кейс 5. Поиск клиента
  Предусловие:
  1. Открыть браузер
  2. Перейти на страницу входа: https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager
  3. Нажать на кнопку customers
  4. В таблице есть клиент с данными: Harry Potter E725JB 1004 1005 1006

  Входные данные:
  1. "Harry"
  2. "Potter"
  3. "E725JB"
  4. "1004"

  Шаги:
  1. В строке поиска, находящейся сверху от таблицы клиентов поочередно ввести слова для поиска из списка
  входных данных

  Ожидаемый результат:
  1. Для каждого слова из входных данных должна найтись как минимум одна запись в таблице
